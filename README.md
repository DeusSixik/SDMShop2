# SDM Shop 2

A completely reworked version of the original mod.

## Stack
- [LDLib](https://www.curseforge.com/minecraft/mc-mods/ldlib) - It was chosen as a replacement for FTB Library due to its capabilities.
- [Сaffeine](https://github.com/ben-manes/caffeine) - Used for storing and managing data
- [Shadow Config](https://www.curseforge.com/minecraft/mc-mods/shadow-config) - Library for creating configs
